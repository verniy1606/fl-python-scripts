# インストール方法 (FL Studio 2024)
Mac: "`/Applications/FL Studio 2024/Contents/Resources/FL/System/Config/Piano roll scripts`" <br>
Windows: "`\FL Studio 2024\System\Config\Piano roll scripts`" <br>
に目的のスクリプトファイル（.pyscript）をコピー
